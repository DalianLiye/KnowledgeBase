# 基础模块 #
- **QtCore**\
提供核心非GUI功能，如信号和插槽、属性、项目模块的基类、序列化等

- **QtGUI**\
提供了基于QtCore的GUI功能：事件、窗口和屏幕、OpenGL和基于光栅的(raster-based)2D绘制以及图像

- **QtWidgets**\
为应用程序提供现成的小部件，包括UI的图形元素

# QML模块和QtQuick模块 #
可以使用以下模块与QML进行交互\
**关于QML语言：** https://doc.qt.io/qt-5.qmlapplications

- **QtQML**\
提供了与QML进行交互的基础接口API

- **QtQuick**\
提供了用于将QtQuick嵌入到Qt应用程序的类

- **QtQuickWidgets**\
提供了用于将QtQuick嵌入到基于widget的应用程序的QQuickWidget类

# 数据可视化模块 #
提供了一些图表，动画等可视化元素

- **QtCharts**\
提供了一些简单可用的图标组件

- **QtDataVisualization**\
提供了一种可以将数据转换为可视化的3D柱状图，散点图等图形的方式

# 多媒体模块 #
提供了声音，录像，以及硬件交互等多媒体相关的解决方案

- **QtMultimedia**\
提供了低层次的多媒体相关功能

- **QtMultimediaWidgets**\
提供了基于widget的多媒体相关的API接口

# WebEngine模块 #
如果项目是基于浏览器或web相关的，可以使用该模块进行交互

- **QtWebEngineWidgets**\
提供了相关widgets用于处理web内容

- **QtWebChannel**\
可以使服务器和客户端之间进行点对点通信(HTML/JavaScript或QML应用程序)
