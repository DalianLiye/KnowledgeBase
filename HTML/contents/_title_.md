# 定义
\<title\>标签定义了文档的标题

# 使用
\<title\>标签在HTML/XHTML文档中是必需的\
**使用效果：**
- 浏览器打开网页时, \<title\>标签的文本内容会显示为TAB页的标题\
- 当网页添加到收藏夹时, \<title\>标签的文本内容会显示在收藏夹中的标题\
- \<title\>标签的文本内容会显示在搜索引擎结果页面

# 示例
```html
<!DOCTYPE html>
<html>
    <head> 
        <meta charset="utf-8"> 
        <title>文档标题</title> <!-- -->
    </head>
    <body>
        文档内容......
    </body>
</html>
```
