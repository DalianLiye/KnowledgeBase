# 定义
\<tbody\>标签用于定义表格的主体

# 使用
主要用于放数据本体, 使代码在语义上结构更加清晰, 没有显示效果上的意义\
标签必须放在\<table\>标签中

# 示例
```html
<table>
    <thead>
        <tr>
            <th>...</th>  
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>...</td>  
        </tr>
        ...
    </tbody>
</table>
```
