HTML文档由各HTML元素(标签)组成\
HTML标签是由尖括号包围的关键词，如\<html\>

# 语法
**语法格式：**\
开始标签 元素内容 结束标签\
**注：**\
开始标签常被称为起始标签(开始标签)，结束标签常称为闭合标签(结束标签)

```html
<p>这是第一个段落。</p>

<!-- 
<p>为开始标签，</p>为结束标签，"这是第一个段落。"为元素内容
->
```

**语法格式说明：**
- 元素以开始标签起始
- 元素以结束标签终止
- 元素的内容是开始标签与结束标签之间的内容
- 某些元素具有空内容(empty content)
- 空元素在开始标签中进行关闭(以开始标签的结束而结束)
- 大多数HTML元素可拥有属性

# 标签种类
## 双标签
HTML标签通常是成对出现的，如\<html\>\</html\>, 称为双标签

## 单标签
有些特殊的标签必须是单个标签(极少情况), 例如\<br /\>,称为单标签

# 元素关系
## 包含关系
包含关系(嵌套)也叫父子关系\
大多数HTML元素可以嵌套(一个元素可以包含其他元素)\
HTML文档由相互嵌套的HTML元素构成
```html
<head>
    <title></title>
</head>
```

## 并列关系
并列关系也叫兄弟关系
```html
<head></head>
<body></body>
```

# 结束标签
关闭标签是可选的, 即使没使用结束标签, 大多数浏览器也会正确地显示\
但最好不要忘记关闭标签, 否则可能会产生不可预料的结果或错误

# 空元素
没有内容的HTML元素被称为空元素, 空元素是在开始标签中关闭的\
\<br\>就是没有关闭标签的空元素(\<br\> 标签定义换行)\
在XHTML、XML以及未来版本的HTML中, 所有元素都必须被关闭\
在开始标签中添加斜杠，比如 \<br/\>, 是关闭空元素的正确方法, HTML、XHTML 和 XML都接受这种方式\
即使\<br\>在所有浏览器中都是有效的, 但使用 \<br/\>其实是更长远的保障

# 大小写
HTML标签对大小写不敏感,许多网站都使用大写的HTML标签
```html
<P>等同于<p>
```
因为万维网联盟(W3C)在HTML4中推荐使用小写, 而在未来(X)HTML版本中强制使用小写
