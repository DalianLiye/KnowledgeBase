# API概述


## API类型
- Alpha	
尝鲜版，包含最新的功能，但不稳定，甚至有Bug\
一般用于短期测试，功能验证

- Beta	
经过很好测试，但细节可能会变动

- Stable	
稳定版，可在正式环境使用


## 访问控制
当通过GUI访问集群时，需要有认证和授权


## 废弃api说明
官方有关于弃用api的说明，具体参照如下：\
URL：https://kubernetes.io/zh-cn/docs/reference/using-api/deprecation-guide/


## Pod与集群
API可以与以下资源进行交互：
- 与运行的Pod交互
- 与节点和集群交互


## 资源类型与别名
pods: po
deployments: deploy
services: svc
namespace: ns
nodes: no
statefulset: sts
daemonset: ds
endpoints: ep
configmap: cm


## 格式化输出
输出json格式	-o json
仅打印资源名称	-o name
以纯文本格式输出所有信息	-o wide
输出yaml格式	-o yaml
