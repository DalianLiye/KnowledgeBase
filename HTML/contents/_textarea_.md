# 定义
\<textarea\>标签用于定义多行文本输入

# 使用
当用户输入内容较多的情况下, 就不能使用文本框表单了, 此时可以使用\<textarea\>标签\
通过\<textarea\>标签可以轻松创建多行文本输入框, 常见于留言板, 评论\
开发中很少使用rows和cols属性, 都是通过CSS来改变大小

# 属性
## rows属性
**定义：**\
用于定义显示的行数\
**值：**\
正整数

## cols属性
**定义：**\
用于定义每行中的字符数\
**值：**\
正整数

# 示例
```html
<textarea rows="3" cols="20">默认文本内容</textarea>
<!--定义文本域，跨3行，每行显示字符20个-->
```
