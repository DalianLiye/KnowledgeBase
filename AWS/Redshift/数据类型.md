# 数字类型 #
## 整数类型 ##
使用SMALLINT、INTEGER 和 BIGINT数据类型存储各种范围的整数

**SMALLINT**
- **说明:** 有符号的二字节整数
- **别名:** INT2
- **存储:** 2 字节
- **范围:** -32768 到 +32767, 即可以表示2的16次方个数字, 正负3万多

**INTEGER**
- **说明:** 有符号的四字节整数
- **别名:** INT 或 INT4
- **存储:** 4 字节
- **范围:** -2147483648 到 +2147483647, 即可以表示2的32次方个数字，正负20多亿

**BIGINT**
- **说明:** 有符号的八字节整数
- **别名:** INT8
- **存储:** 8 字节
- **范围:** -9223372036854775808 到 9223372036854775807, 即可以表示2的64次方个数字

**注:**\
数值超过各自类型规定的范围会报错

## 浮点类型 ##
使用REAL和DOUBLE PRECISION数据类型可存储具有可变精度的数值\
这些类型是不精确的类型，意味着一些值是作为估计值存储的\
因此存储和返回某个特定值可能导致细微的差异\
如果需要精确的存储和计算（如货币金额），请使用DECIMAL数据类型

**REAL**
- **说明:** 单精度浮点数
- **别名:** FLOAT4
- **存储:** 4字节
- **范围:** 精度为8个有效位(至少6位)

**DOUBLE PRECISION**
- **说明:** 双精度浮点数
- **别名:** FLOAT8 或 FLOAT
- **存储:** 8字节
- **范围:** 精度为17个有效位(至少15位)
