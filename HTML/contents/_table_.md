# 定义
\<table\>为表格标签, 用来定义网页中的表格

# 使用
表格主要用于显示, 展示数据, 因为它可以让数据显示的非常的规整, 使其可读性好\
表格不是用来布局页面的, 是用来展示数据的

# 示例
```html
<table align="center" border="1"  cellpadding="1" cellspacing="0"  width="100" height="50">
    <tr>
        <th>...</th>  
    </tr>
    <tr>
        <td>...</td>  
    </tr>
    ...
</table>
```

# 属性
**注:**\
表格标签这部分属性在实际开发中不常用, 后面通过CSS来设置

## align属性
**定义：**\
规定表格相对周围元素的对齐方式\
**值：**\
固定值left,center,right
```html
<table align='center' >
.....
</table>   
<!--表格在页面中央显示-->
```

## border属性
**定义：**\
规定表格单元是否拥有边框, 默认为"", 表示没有边框\
**值：**\
像素值
```html
<table border="1" >
.....
</table>  
```

## cellpadding属性
**定义：**\
规定单元格边沿与其内容之间的空白,默认1像素\
**值：**\
像素值

## cellspacing属性
**定义：**\
规定单元格之间的空白, 默认2像素\
**值：**\
像素值

## width属性
**定义：**\
规定表格的宽度\
**值：**\
像素值或百分比

## height属性
**定义：**\
规定表格的高度\
**值：**\
像素值或百分比

# 表格头和表格主体
因为表格可能很长, 为了更好的表示表格的语义, 可以将表格分割成表格头和表格主体两大部分\
在表格标签中, 分别用\<thead\>标签表格的头部区域, \<tbody\>标签表格的主体区域, 这样可以更好的分清表格结构\
具体请参照\<thead\>标签和\<tbody\>标签
