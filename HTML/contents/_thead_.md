# 定义
\<thead\>标签用于定义表格的头部

# 使用
主要用于放表格头, 使代码在语义上结构更加清晰, 没有显示效果上的意义\
标签必须放在\<table\>标签中, 内部必须拥有\<tr\>标签, 一般是位于第一行

# 示例
```html
<table>
    <thead>
        <tr>
            <th>...</th>  
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>...</td>  
        </tr>
        ...
    </tbody>
</table>
```
