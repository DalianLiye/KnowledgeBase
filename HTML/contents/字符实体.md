# 定义
在HTML页面中, 一些特殊的符号很难或者不方便直接使用, 此时就可以使用字符实体来替代\
这些字符是HTML预留的, 类似于java中的关键字, 开发者是不能直接使用的\
比如不能使用小于号（<）和大于号（>）, 这是因为浏览器会误认为它们是标签

# 使用
**字符实体格式**
```html
&entity_name;
或
&#entity_number;
```

**注：**
- 当显示小于号, 必须写成："&lt;" 或 "&#60;" 或 "&#060;"\
使用实体名(比如&lt)而不是数字(比如&#60;或 &#060;)的好处是名称易于记忆, 坏处是浏览器也许并不支持所有实体名称(对实体数字的支持却很好)
- 实体名称对大小写敏感

# 不间断空格
HTML中的常用字符实体是不间断空格(&nbsp;), 英文全称Non-breaking Space\
浏览器总是会截短HTML页面中的空格\
如果您在文本中写10个空格, 在显示该页面之前, 浏览器会删除它们中的9个\
如需在页面中增加空格的数量, 需要使用&nbsp; 字符实体


# 常用的字符实体
重点记住以下几个特殊字符：
- 空格 ：\&nbsp;
- 大于号 ：\&gt;
- 小于号 ：\&lt;
