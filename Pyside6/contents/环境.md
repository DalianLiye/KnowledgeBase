# 安装 #
打开cmd，进入python安装路径，或者打开anaconda终端，执行以下命令
- **安装默认版本**
    ```cmd
    pip install PySide2
    ```
- **安装指定版本**
    ```cmd
    pip install PySide2==5.15.0
    ```
    或
    ```cmd
    pip install --index-url=http://download.qt.io/snapshots/ci/pyside/5.15/latest pyside2 --trusted-host download.qt.io
    ```
- **测试安装**\
    创建python文件(比如test.py)，执行以下代码，正常打印版本号，说明安装成功
    ```python
    import PySide2.QtCore

    # Prints PySide2 version
    print(PySide2.__version__) #输出：5.15.2.1

    # Prints the Qt version used to compile PySide2
    print(PySide2.QtCore.__version__) #输出：5.15.2
    ```
