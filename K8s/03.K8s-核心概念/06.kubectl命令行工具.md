# kubectl命令行工具
K8s提供的kubectl，是使用K8s API与K8s集群的控制平面进行通信的命令行工具\
具体命令，参考以下文档\
URL：https://kubernetes.io/docs/reference/generated/kubectl/kubectl-commands


## 资源操作
kubectl命令行工具涉及资源的操作，主要包含以下几种类型:
- 创建对象
- 显示和查找资源
- 更新资源
- 修补资源
- 编辑资源
- scale资源
- 删除资源

注：具体更多命令请参照官方文档，或使用以下语句查询
```shell
kubectl <opt_type> --help
```
