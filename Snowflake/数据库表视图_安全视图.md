# 安全视图概述


## 非安全视图弊端
- 内部优化可能会间接公开数据
- 视图定义对其他用户可见


# 何时使用安全视图
当视图专门指定用于数据隐私时，应将其定义为安全视图（即限制对不应向基础表的所有用户公开的敏感数据的访问权限）

安全视图主要应用于数据安全

安全视图的性能可能比非安全视图慢\
在决定是否使用安全视图时，应考虑视图的用途，并考虑数据隐私/安全性与查询性能之间的权衡

# 创建安全视图
要创建安全视图，请在 CREATE VIEW 或 CREATE MATERIALIZED VIEW 命令中指定 SECURE 关键字\
要将现有视图转换为安全视图并转换回常规视图，请在 ALTER VIEW 或 ALTER MATERIALIZED VIEW 命令中设置/取消设置 SECURE 关键字
