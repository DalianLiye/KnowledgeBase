# 语法 #
- **文件名**\
    文件后缀名为.g4
    ```java
    XXXXXX.g4   
    ```
- **文件内容**
    ```java
    grammmar XXXXXX  //grammar后的XXXXX必须要跟文件名一致

    /*文法规则1，及其分支
    各个分支用|分割 
    各规则用分号结束
    命名规则：字母小写*/
    rule1: ...... 
        | .....
        | .....
        ...
        ; 
    /*文法规则2，及其分支*/
    rule2: ......
        | .....
        | .....
        ...
        ;
    ...

    /*词法规则1，它没有分支
    命名规则：字母大写*/
    TOKEN1: .....;
    /*词法规则2*/
    TOKEN2: .....;
    ...

    /*定义哪些字符是不需要识别的*/
    WS: [ \t\r\n]+ -> skip; 
    ```

语法包含了一系列描述语言结构的规则，这些规则既包括描述语法结构的规则，也包括描述词汇符号(词法符号)的规则

语法分析器的规则以小写字母开头

词法分析器的规则以大写字母开头

使用|来分隔同一个语言规则的若干备选分支，使用圆括号把一些符号组成子规则,比如('*'|'/')













*：匹配以任意个，包括0个，1个或任意多个
+: 匹配至少1个，包括1个，或任意多个,但不能是0个
