# 环境配置(python版本) #
- **下载antlr4**
  ```cmd
  pip install antlr4-python3-runtime
  ```
- **配置PyCharm**\
  按照如下顺序，在PyCharm上安装antlr4插件\
  1) File --> Settings --> Plugins --> 搜索antlr4 ==> 安装后重启PyCharm\
  
  2) 右键点击.g4文件 --> Configure ANTLR.. --> Language里选择Python

  操作之后，就可以在PyCharm上直接利用插件编译调试antlr语法文件

  **注：**\
  插件的版本和通过pip安装的antlr4版本不要相差太多\
  遇到下面错误，暂时可以忽略,也可以通过从github上下载4.10.1版本的脚本并拷贝至.....\Lib\site-packages\antlr4\文件夹下
  ```
  ANTLR runtime and generated code versions disagree: 4.11.1!=4.10.1
  ```

- **其他**\
  如果不想通过PyCharm来生成文件，则可通过以下方式\
  参考URL：https://github.com/jszheng/py3antlr4book
